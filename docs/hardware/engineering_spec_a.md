# 工程规格推导 A：硬件规格量化

> 基于物理公式推导，量化具身机器人双目视觉模组的硬件参数

---

## 1. 核心方程：深度精度

双目视觉深度公式：
```
Z = (f × B) / d

δZ = Z² × δd / (f × B)  [误差传播公式]
```

| 符号 | 含义 | 单位 |
|------|------|------|
| Z | 深度距离 | m |
| f | 焦距（等效35mm全画幅） | mm |
| B | 基线距离 | m |
| d | 视差 | pixel |
| δd | 视差测量误差 ≈ 1 pixel（亚像素细化后 ≈ 0.25 pixel）| pixel |

## 3. 镜头选型推导

### 3.1 靶面与 CRA 匹配

镜头 CRA 必须 ≤ 传感器 CRA（0° + 5°容差 = **5°**）
选型要求：
- S-mount / M12 镜头座（成本最低，批量 $1.5~$3）
- CRA ≤ 5°（机械后焦可调）
- IR-CUT 镀膜（940nm NIR 用于低光 VIO）

### 3.2 FOV 推导

水平视场角公式：
```
FOV_H = 2 × arctan(H_sensor / (2 × f_eq))

其中 H_sensor = 7.68mm（IMX678水平）
     f_eq = 焦距 / Crop_Factor
     Crop_Factor = 8.84mm / 43.27mm（全画幅对角线）= 0.204
     → f_eq = f_mm / 0.204
```

| 焦距 | 等效35mm焦距 | 水平FOV | 垂直FOV | 评价 |
|------|------------|--------|--------|------|
| 2.8mm | ~13.7mm | **109°** | 61° | 太广，畸变大 |
| 4.0mm | ~19.6mm | **88°** | 53° | 较广 |
| **5.5mm** | ~27.0mm | **71°** | 42° | ✅ 推荐 |
| 7.0mm | ~34.3mm | 58° | 34° | 视场偏窄 |

**选择：5.5mm M12 镜头（FOV ≈ 70°H）**

### 3.3 景深估算

```
景深公式：DoF = 2 × u² × δ × N / (f²)
其中：
  u = 对焦距离 (m)
  δ = 弥散圆直径 ≈ 0.03mm（IMX678要求）
  N = 光圈数 (F#)
  f = 焦距 = 5.5mm
```

| 对焦距离 | F/2.0 景深 | F/2.8 景深 | 适用场景 |
|---------|-----------|-----------|---------|
| 0.5m | 0.41~0.63m | 0.36~0.82m | 近距操作 |
| 1.0m | 0.76~1.35m | 0.63~2.1m | AMR 抓取 |
| 2.0m | 1.4~3.1m | 1.1~5.5m | 导航避障 |

**选择 F/2.0**（大光圈 = 更多进光量 = 低光性能更好）

## 4. 基线距离（Baseline）推导

### 4.1 深度量程约束

最大视差 = **num_disparities × 16**（SAD Windows匹配）

| num_disparities | 最大视差(像素) | 最小深度 @ 5.5mm | 最大深度 |
|----------------|--------------|-----------------|---------|
| 64 | 1024 | 0.22m | 8.8m |
| **128** | 2048 | **0.44m** | **17.6m** |
| 192 | 3072 | 0.66m | 26.4m |

**选择 num_disparities = 128**（平衡精度与计算量）

### 4.2 深度精度 vs 基线关系

```
δZ = Z² / (f_pix × B)  [假设亚像素细化 δd = 0.25 pixel]

f_pix = f_eq / 像素尺寸
      = 27.0mm / 2.0μm = 13500 pixels
```

| 基线 | 0.5m 精度 | 1m 精度 | 2m 精度 | 5m 精度 |
|------|---------|--------|--------|--------|
| 50mm | 1.4cm | 5.5cm | 22cm | 138cm |
| **80mm** | **0.9cm** | **3.4cm** | **14cm** | **86cm** |
| 120mm | 0.6cm | 2.3cm | 9.2cm | 57cm |
| 200mm | 0.4cm | 1.4cm | 5.5cm | 34cm |

**结论**：
- **80mm 基线**是成本/精度的最优平衡点
- 0.5m 内精度 < 1cm → 精密抓取足够
- 5m 处精度 86cm → 导航避障勉强够（需要语义辅助）
- 如需 5m 精度 < 30cm → 需 200mm 基线（结构空间挑战大）

### 4.3 结构约束

```
B = 80mm ± 0.5mm（机械公差）

总宽度估算：
  左右各 1个 IMX678 PCB（含FPC）≈ 12mm
  中间 DESERIALIZER 板 ≈ 30mm
  结构外壳 ≈ 25mm
  → 模组总宽 ≈ 80 + 12 + 30 + 25 = 147mm
  → 适配人形机器人头部（典型 150~180mm）
```

### 5.1 规格表

| 参数 | 值 | 规格 |
|------|-----|------|
| 类型 | 6轴（3轴 gyro + 3轴 accel）| |
| 陀螺仪 range | ±2000°/s | 车规级 |
| 陀螺仪 noise density | **0.014°/s/√Hz** | 业界顶级 |
| 陀螺仪 rate | 200Hz | SPI/I2C |
| 加速度计 range | ±16g | |
| 加速度计 noise density | **400 μg/√Hz** | |
| 加速度计 rate | 400Hz（高速）| |
| I2C 地址 | ACC: 0x18, GYRO: 0x68 | |
| 温度范围 | **-40~105°C** | 车规 |

### 5.2 VIO 精度估算

陀螺仪积分角度噪声：
```
σ_θ = σ_gyro × √(BW) × √(3) × t_int

σ_gyro = 0.014°/s/√Hz
BW = 100Hz（低通滤波后）
t_int = 1 frame = 33ms（@ 30fps）

→ σ_θ ≈ 0.014 × √100 × 1.732 × 0.033
  ≈ 0.24°/s × 0.033s ≈ 0.008°
```

位置误差传播（纯积分）：
```
σ_pos = 0.5 × σ_acc × t²
σ_acc = 400 μg/√Hz × √100 × 1.732 ≈ 6.9 mg ≈ 0.068 m/s²

1帧（33ms）：σ_pos ≈ 0.5 × 0.068 × 0.033² ≈ 3.7 μm
1秒（30帧累加）：σ_pos ≈ 0.068 × 0.5 × 1² ≈ 34mm/s → 累积发散

→ 必须依赖视觉更新修正（EKF 融合）
```

**BMI088 + IMX678 组合**：视觉更新频率 30Hz
→ VIO 位置精度 ≈ **深度 × 0.5%**（典型值）
→ 1m 处 ≈ 5mm RMS

### 6.3 透明物体抓取场景验证

目标场景：仓储 AMR 抓取透明塑料袋

```
最坏情况深度误差预算：
  目标距离：Z = 0.5m
  要求：δZ < 0.5cm（±0.5cm 定位精度）

计算验证（80mm基线，5.5mm焦距）：
  δZ = Z² × δd / (f × B)
     = 0.5² × 0.25 / (0.0055 × 0.08)
     = 0.25 × 0.25 / 0.00044
     = 142 pixel → δZ = 0.5²/0.0142 = 17.6cm ← 不够！

改进方案：
  1. 缩短焦距至 3.5mm → f = 17.2mm eq → δZ = 2.6cm ✅
  2. 增大基线至 120mm → δZ = 2.0cm ✅

结论：透明物体精密抓取（0.5m内）推荐：
  - 基线 120mm（高精度版）
  - 或焦距 3.5mm（广角版）
```
