**方案 A vs B 对比**：

| 规格 | IMX678（¥274）| IMX264（¥174）| 差异 |

| 分辨率 | 4K | 2K | +100% |

**决策**：选用 **IMX678**（¥274 BOM），但 IMX264 方案备用（¥174 BOM，¥506 总BOM↓）

## 6. 成本降低路线图

### 6.1 第一年降本路径

| 时间 | BOM 目标 | 措施 |
|------|---------|------|
| 0~3月 | ¥584 | 方案A（IMX678）|
| 3~6月 | ¥520 (-11%) | 优化结构（压铸→注塑），减少开模 |
| 6~9月 | ¥465 (-20%) | IMX264 备用方案切换 |
| 9~12月 | ¥420 (-28%) | 定制 Sensor 芯片（裸片 COB）|

### 6.2 关键降本杠杆

| 杠杆 | 潜力 | 风险 |
|------|------|------|
| IMX678 → IMX264 | -¥100 | 精度下降，帧率降 |
| MAX96712 → 替代串行器 | -¥20 | 距离缩短 |
| 铝合金 CNC → 注塑 | -¥20 | 初期模具费高 |
| 铝合金 → 塑料外壳 | -¥25 | 散热变差 |
| 采购量 1K → 10K | -15% 器件 | 库存风险 |
| 国产替代（IMU/SerDes）| -¥30 | 可靠性验证 |
