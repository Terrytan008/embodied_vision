# 发票分类与汇总

按类别自动分类发票文件，生成 Excel 汇总表。

## 主分类规则

| 类别 | 关键词 |
|------|--------|
| 交通出行 | 交通、出行、打车、滴滴、出租车、火车票、机票、航空、地铁、公交、高速、ETC、汽油、充电、uber、didi、高德、行程、停车费、过路费、船票、船运、阳光出行 |
| 餐饮美食 | 餐饮、餐费、餐服务、饭店、餐厅、午餐、晚餐、美食、外卖、奶茶、咖啡、酒楼 |
| 办公用品 | 办公、文具、打印、耗材、电脑、设备、配件、纸张、墨盒、office、半导体、电子元件、立创 |
| 酒店住宿 | 酒店、住宿、宾馆、旅馆、民宿、房费、hotel、inn、room |

## 发票 vs 行程单过滤规则

- **国内人民币行程单** → 排除（不纳入汇总表）
  - 关键词：`行程报销单`、`电子行程单`、`行程单`
  - 例：`滴滴出行行程报销单_01.pdf`、`高德打车电子行程单_11.pdf`
- **外币（USD/HKD等）行程单** → 保留（国外出差报销需要）
  - 例：`【13.35USD - 1个行程】高德打车电子行程单_04.pdf`
- **电子发票（含"电子发票"/"电子普通发票"）** → 保留

## 去重逻辑

| 发票类型 | 去重方式 |
|----------|----------|
| 有发票代码的中国正规发票 | 签名去重（发票代码+日期+金额） |
| 无发票号的境外文件（Uber等） | MD5 内容哈希去重 |
